# Структура проекта "Шахматы"

```python
ChessGame/
├── src/
│   ├── board/
│   │   ├── Board.java # логика доски
│   ├── pieces/
│   │   ├── Piece.java # абстрактный класс для всех фигур
│   │   ├── King.java # класс для короля
│   │   ├── Queen.java # класс для ферзя
│   │   ├── Rook.java # класс для ладьи
│   │   ├── Bishop.java # класс для слона
│   │   ├── Knight.java # класс для коня
│   │   ├── Pawn.java # класс для пешки
│   ├── game/
│   │   ├── ChessGame.java # основная логика игры
│   ├── ui/
│   │   ├── ChessUI.java # графический интерфейс
```
