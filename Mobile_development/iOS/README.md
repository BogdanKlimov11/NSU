Структура проекта "Змейка"

```python
SnakeGame/
│── Controllers/
│   ├── ViewController.swift // основной контроллер для отображения игры и взаимодействия с пользователем.
│── Models/
│   ├── GameModel.swift // модель игры, отвечающая за логику змейки, еды, движения и столкновений.
│   ├── ScoreManager.swift // класс для управления очками и сохранения результата.
│── Views/
│   ├── GameView.swift // кастомный view для отображения игры: змейка, еда, поле.
├── AppDelegate.swift // основной файл, который инициализирует приложение.
```
