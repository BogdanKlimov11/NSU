# Структура проекта "Крестики-нолики"

```python
TicTacToeApp/
│── app/
│   ├── src/
│   │   ├── main/
│   │   │   ├── res/
│   │   │   │   ├── layout/
│   │   │   │   │   ├── activity_main.xml # главное меню
│   │   │   │   │   ├── activity_game.xml # игровое поле

│   │   │   │   │   ├── colors.xml # цвета для приложения
│   │   │   │   │   ├── styles.xml # стили для приложения
│   │   │   ├── java/com/tictactoe/  
│   │   │   │   ├── MainActivity.kt # главное меню  
│   │   │   │   ├── GameActivity.kt # управление состояниями
│   │   │   │   ├── TicTacToeViewModel.kt # управления логикой игры
```
